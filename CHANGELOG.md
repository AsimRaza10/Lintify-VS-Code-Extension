# Changelog

## 1.0.0 (Professional Release)
### ✨ New Features
- **Professional Code Review**: Structured, easy-to-read review output with clear sections
- **Accept/Reject Workflow**: Preview changes and choose to accept or reject before applying
- **Add Code Command**: Generate and insert AI-suggested code with preview
- **Review Whole File**: Review entire files, not just selections
- **Diff View**: Side-by-side diff before applying code fixes
- **Welcome Page**: Helpful onboarding for new users
- **Graceful Language Handling**: Warns for unsupported languages

### 🔧 Improvements
- **Fixed Editor Focus Issues**: Properly handles editor focus after diff view
- **Restricted to Gemini 1.5 Flash**: Uses only the most reliable model
- **Enhanced Error Handling**: Better error messages and user feedback
- **Output Channel**: Dedicated "Lintify" output for logs and troubleshooting
- **Undo Support**: All changes are undoable (Ctrl+Z)
- **Professional Branding**: PNG emoji icon and modern UI
